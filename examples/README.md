# Example Outputs

These are real outputs generated by running each workflow against the mock data in `data/`. Every result was produced by Claude via the Anthropic SDK — no manual editing.

## What's Here

| File | Workflow | What It Shows |
|------|----------|---------------|
| `press_digest.md` | Daily Press Digest | 17 articles scored for relevance, classified by topic/sentiment, ranked and formatted for morning distribution |
| `rapid_response_alerts.json` | Rapid Response | 8 events triaged into P0/P1/P2 tiers with talking points, escalation guidance, and owner routing |
| `briefing.md` | Briefing Generator | Spokesperson prep document for a Dario Amodei CNBC interview — key messages, anticipated questions, bridging language |
| `pull_through_report.md` | Pull-Through Tracker | Message fidelity analysis across 18 articles measuring how well Anthropic's key narratives landed in earned media |
| `internal_comms_report.md` | Internal Comms | Full draft-review-format pipeline for an all-hands update — includes editorial scoring, sensitivity flags, and Slack/email formatting |

## Reproducing These

```bash
source .venv/bin/activate
python scripts/run_press_digest.py
python scripts/run_rapid_response.py
python scripts/run_briefing.py
python scripts/run_pull_through.py
python scripts/run_internal_comms.py
```

Outputs land in `outputs/` (gitignored). These examples are snapshots checked in for reference.
